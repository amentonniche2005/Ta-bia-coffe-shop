Mathématiques Industrielles

Le moteur de Stock
ne ment jamais.

Sarbini ne se contente pas de faire "-1" quand vous vendez une pizza. Son dictionnaire de conversion embarqué détruit les erreurs humaines liées aux unités de mesure.

  • Conversion Universelle Automatique Vous achetez un sac de Fromage en Kilos (kg). Votre supplément est configuré à 50 Grammes (g). Le serveur convertit et déduit très exactement 0.050 de votre base de données.
  • Bouclier Anti-Vente Négative Le client commande 3 Burgers. L'algorithme simule virtuellement le panier. S'il détecte qu'il ne reste que 2 pains en stock, il bloque l'ajout de la 3ème unité à la caisse instantanément.
  • Audit Physique (Chiffrage des vols) Mode Inventaire. Tapez ce que vous voyez sur l'étagère. L'ERP calcule l'écart avec la base, le multiplie par le Prix d'Achat, et vous montre l'impact de la perte en Dinars.
server/controllers/erpEngine.js

// 1. Le dictionnaire d'Intelligence Artificielle

const CONVERSIONS = {

'mg': 0.001, 'g': 1, 'kg': 1000,

'ml': 1, 'cl': 10, 'L': 1000,

'cac': 5, 'cas': 15 // Cuillères cuisine

};


// 2. Exécution lors du paiement en Caisse

function calculerQuantiteDestockage(qte, uRecette, uStock) {

const fSource = CONVERSIONS[uRecette];

const fCible = CONVERSIONS[uStock];

return (qte * fSource) / fCible;

}
